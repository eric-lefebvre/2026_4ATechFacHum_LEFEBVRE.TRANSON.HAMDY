# Description du projet:

Jeu de golf

# Capteurs nécessaires:
  - Accéléromètre 3 axes (ACC)
  - ElectroMyoGramme (EMG)
  - ElectroDermal Activity (EDA)
  - ElectroCardioGramme (ECG)

# Rôle de chaque capteur:
  - AAC -> Visée
  - EMG -> Tir
  - EDA ou ECG -> Capte le stress
